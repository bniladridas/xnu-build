What is XNU?
  X = "X is Not Unix"
  Combines Mach (microkernel) + BSD (POSIX) + I/O Kit (drivers)
  This is the macOS kernel

What will you build?
  - Complete XNU kernel compilation
  - Mach microkernel + BSD layer + I/O Kit
  - Debug symbols for analysis
  - Userland tools and libraries

Can I use this on my Mac?
  No - Modern macOS prevents kernel replacement (SIP)
  Yes - Test in virtual machines instead

Why do this?
  - Learn how operating systems work
  - Understand kernel architecture
  - Study real production code
  - Develop kernel engineering skills
